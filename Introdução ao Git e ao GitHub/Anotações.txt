O Git Bash é um terminal estendido para otimizar o uso do Git.
 
 - Para adicionar uma nova informação da nossa maquina local direto para o
servidor remoto do Git Hub, basta dar os seguintes comandos:
( git status--vai mostar o status da pasta nova )

## git add.  ou um git add -A ## 
 - adicionará a pasta da nossa maquina para o 
controle de versão local do Git Hub.

## git commit -m "mensagem desejada" ##
 - será criado um commit local.
para adicionalo ao repositório remoto deve-se dar:

## git push origin main ##
 - Depois disso ao dar um F5 ou atualizar o seu Git Hub aparecera suas atualizações.
